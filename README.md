# ⚡ Universal Flipkart Live Scraper Dashboard

An interactive, high-performance SaaS-like web application built using **Python, Streamlit, and Selenium**. This tool allows e-commerce brand owners and businesses to extract real-time product data, current pricing, and high-quality images directly from Flipkart instantly.

---

## ✨ Key Features
* 🔍 **Real-Time Live Scraping:** Dynamically fetches live data based on whatever keyword the user types (e.g., Jeans, Chairs, Smartphones).
* 🎛️ **Premium Dark UI/UX:** A highly interactive dashboard with responsive product hover cards and smooth CSS glowing transitions.
* 📊 **4-Column Grid Showcase:** Clean alignment of scraped product details including images, titles, and real-time accurate prices.
* 📥 **Instant Dataset Download:** Feature to export the scraped data directly into a structured `.CSV` file with a single click.
* 🤖 **Headless Browser Execution:** Runs seamlessly in the background without interrupting the user experience.

---

## 🛠️ Tech Stack & Architecture
* **Frontend Dashboard:** Streamlit (Python Core Components)
* **Web Scraping Engine:** Selenium WebDriver (Automated Browser Manipulation)
* **Driver Management:** Webdriver-manager (Automates setup for Chrome Binary)
* **Data Formatting:** Pandas (For structured CSV framing & state holding)
* **Styling Layer:** Custom CSS injection for layout locking & hover dynamics
